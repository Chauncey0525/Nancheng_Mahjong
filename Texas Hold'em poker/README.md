# 德州扑克小游戏

一个基于HTML、CSS和JavaScript开发的德州扑克小游戏，可以在浏览器中直接运行。

## 功能特点

- 🎮 完整的德州扑克游戏流程（翻牌前、翻牌圈、转牌圈、河牌圈、摊牌）
- 🃏 52张标准扑克牌
- 🤖 智能AI对手
- 💰 筹码系统和下注机制
- 🎨 美观的现代化界面
- 📱 响应式设计，支持移动设备

## 游戏规则

### 基本规则
- 每位玩家发2张底牌
- 公共牌依次发出：翻牌圈3张、转牌圈1张、河牌圈1张
- 玩家可以使用2张底牌和5张公共牌中的任意5张组成最佳手牌

### 手牌等级（从高到低）
1. 皇家同花顺（A-K-Q-J-10，同花色）
2. 同花顺（五张连续同花色）
3. 四条（四张相同点数）
4. 葫芦（三条+一对）
5. 同花（五张同花色）
6. 顺子（五张连续）
7. 三条（三张相同点数）
8. 两对
9. 一对
10. 高牌

### 操作说明
- **下注**：输入金额并点击"下注"按钮
- **过牌**：当无需跟注时可以选择过牌
- **弃牌**：放弃当前手牌
- **全押**：将所有筹码下注

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 游戏会自动开始
3. 根据游戏提示进行操作

## 注意事项

- 建议使用现代浏览器（Chrome、Firefox、Edge等）
- 游戏数据存储在浏览器内存中，刷新页面会重置游戏
- AI对手会根据手牌强度做出决策
