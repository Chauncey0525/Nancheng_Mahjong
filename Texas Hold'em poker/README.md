# 德州扑克小游戏

一个基于HTML、CSS和JavaScript开发的德州扑克小游戏，可以在浏览器中直接运行。

## 功能特点

- 🎮 完整的德州扑克游戏流程（翻牌前、翻牌圈、转牌圈、河牌圈、摊牌）
- 🃏 52张标准扑克牌
- 👥 **多人游戏模式**：支持2-6人同时游戏
- 🎯 **单人模式**：经典的1v1对战
- 🤖 智能AI对手，根据手牌强度做出决策
- 💰 筹码系统和下注机制（盲注、下注、跟注、加注、全押）
- ⚙️ **游戏设置**：可自定义初始筹码、盲注大小、最小加注倍数等
- 🔊 **音效系统**：下注、弃牌、获胜等操作配有音效（可开关）
- 🎬 **动画效果**：卡牌发牌、筹码动画、玩家高亮等（可开关）
- 📊 **游戏历史记录**：自动记录每局游戏详情，支持查看和清空
- 🎨 美观的现代化界面
- 📱 响应式设计，支持移动设备
- 🔄 玩家轮转和庄家系统
- 💬 **增强的用户反馈**：实时输入验证、错误提示、成功提示等

## 游戏规则

### 基本规则
- 每位玩家发2张底牌
- 公共牌依次发出：翻牌圈3张、转牌圈1张、河牌圈1张
- 玩家可以使用2张底牌和5张公共牌中的任意5张组成最佳手牌

### 手牌等级（从高到低）
1. 皇家同花顺（A-K-Q-J-10，同花色）
2. 同花顺（五张连续同花色）
3. 四条（四张相同点数）
4. 葫芦（三条+一对）
5. 同花（五张同花色）
6. 顺子（五张连续）
7. 三条（三张相同点数）
8. 两对
9. 一对
10. 高牌

### 操作说明
- **下注**：输入金额并点击"下注"按钮（必须至少等于当前下注，加注需满足最小加注倍数）
- **跟注**：点击"跟注"按钮，自动补齐到当前下注尺度
- **过牌**：当无需跟注时可以选择过牌
- **弃牌**：放弃当前手牌，退出本轮游戏
- **全押**：将所有筹码下注

### 下注规则
- **跟注**：必须至少等于当前下注尺度
- **加注**：必须至少等于当前下注 × 最小加注倍数（默认2倍）
  - 例如：当前下注285，最小加注为285 × 2 = 570
- **全押**：允许不足最小加注（特殊情况）

### 多人模式特点
- 支持2-6人同时游戏
- 庄家位置随机分配，每轮自动轮转
- 小盲注和大盲注系统
- 玩家按顺序行动，直到所有玩家下注相等
- 摊牌时比较所有未弃牌玩家的手牌
- 支持平局情况（多个玩家手牌相同）

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 选择游戏模式：
   - **单人模式**：与1个AI对手对战
   - **多人模式**：选择2-6人进行游戏（你+多个AI对手）
3. 游戏开始后，根据提示进行操作
4. 等待你的回合时，可以观察其他玩家的行动

### 游戏设置

点击"设置"按钮可以自定义：
- **初始筹码**：每位玩家的起始筹码（100-10000）
- **小盲注**：小盲注金额（5-500）
- **大盲注**：大盲注金额（10-1000）
- **最小加注倍数**：加注时的最小倍数（2倍、3倍或自定义1-10倍）
- **音效开关**：启用/禁用游戏音效
- **动画开关**：启用/禁用动画效果

设置会自动保存到浏览器本地存储。

### 游戏历史

点击"游戏历史"按钮可以：
- 查看所有游戏记录
- 查看每局游戏的详细信息（玩家、手牌、结果等）
- 清空历史记录

历史记录会自动保存到浏览器本地存储。

## 注意事项

- 建议使用现代浏览器（Chrome、Firefox、Edge等）
- 游戏数据存储在浏览器内存中，刷新页面会重置游戏（但设置和历史记录会保留）
- AI对手会根据手牌强度做出决策
- 多人模式下，当前行动的玩家会有绿色高亮显示
- 已弃牌的玩家会显示灰色，并标注"已弃牌"
- 全押的玩家会显示橙色标识
- 游戏过程中无法看到电脑的手牌和手牌类型（保持公平性）
- 输入框会实时验证下注金额，错误时会显示红色边框提示

## 游戏界面说明

- **当前行动玩家**：绿色边框高亮显示
- **已弃牌玩家**：灰色半透明，显示"已弃牌"标识
- **全押玩家**：显示"全押"橙色标识
- **庄家位置**：玩家名称后显示"(庄)"
- **盲注位置**：玩家名称后显示"(小盲)"或"(大盲)"

## 交互反馈

- **实时输入验证**：输入下注金额时，会实时显示验证状态
  - 红色边框：金额不足或超出筹码
  - 橙色边框：金额不足最小加注
  - 正常状态：金额有效
- **错误提示**：操作失败时会显示详细的错误信息，包含具体数值
- **成功提示**：保存设置等操作成功时会显示成功消息并自动关闭窗口
- **动画反馈**：错误消息有闪烁动画，成功消息有脉冲动画
