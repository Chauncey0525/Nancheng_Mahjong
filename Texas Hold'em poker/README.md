# 德州扑克小游戏

一个基于HTML、CSS和JavaScript开发的德州扑克小游戏，可以在浏览器中直接运行。

## 功能特点

- 🎮 完整的德州扑克游戏流程（翻牌前、翻牌圈、转牌圈、河牌圈、摊牌）
- 🃏 52张标准扑克牌
- 👥 **多人游戏模式**：支持2-6人同时游戏
- 🎯 **单人模式**：经典的1v1对战
- 🤖 智能AI对手，根据手牌强度做出决策
- 💰 筹码系统和下注机制（盲注、下注、跟注、加注、全押）
- 🎨 美观的现代化界面
- 📱 响应式设计，支持移动设备
- 🔄 玩家轮转和庄家系统

## 游戏规则

### 基本规则
- 每位玩家发2张底牌
- 公共牌依次发出：翻牌圈3张、转牌圈1张、河牌圈1张
- 玩家可以使用2张底牌和5张公共牌中的任意5张组成最佳手牌

### 手牌等级（从高到低）
1. 皇家同花顺（A-K-Q-J-10，同花色）
2. 同花顺（五张连续同花色）
3. 四条（四张相同点数）
4. 葫芦（三条+一对）
5. 同花（五张同花色）
6. 顺子（五张连续）
7. 三条（三张相同点数）
8. 两对
9. 一对
10. 高牌

### 操作说明
- **下注**：输入金额并点击"下注"按钮（必须至少等于当前下注）
- **过牌**：当无需跟注时可以选择过牌
- **弃牌**：放弃当前手牌，退出本轮游戏
- **全押**：将所有筹码下注

### 多人模式特点
- 支持2-6人同时游戏
- 庄家位置随机分配，每轮自动轮转
- 小盲注和大盲注系统
- 玩家按顺序行动，直到所有玩家下注相等
- 摊牌时比较所有未弃牌玩家的手牌
- 支持平局情况（多个玩家手牌相同）

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 选择游戏模式：
   - **单人模式**：与1个AI对手对战
   - **多人模式**：选择2-6人进行游戏（你+多个AI对手）
3. 游戏开始后，根据提示进行操作
4. 等待你的回合时，可以观察其他玩家的行动

## 注意事项

- 建议使用现代浏览器（Chrome、Firefox、Edge等）
- 游戏数据存储在浏览器内存中，刷新页面会重置游戏
- AI对手会根据手牌强度做出决策
- 多人模式下，当前行动的玩家会有绿色高亮显示
- 已弃牌的玩家会显示灰色，并标注"已弃牌"
- 全押的玩家会显示橙色标识

## 游戏界面说明

- **当前行动玩家**：绿色边框高亮显示
- **已弃牌玩家**：灰色半透明，显示"已弃牌"标识
- **全押玩家**：显示"全押"橙色标识
- **庄家位置**：玩家名称后显示"(庄)"
- **盲注位置**：玩家名称后显示"(小盲)"或"(大盲)"
