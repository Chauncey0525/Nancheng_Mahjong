# 古代皇帝评分排名系统

一个基于多维度指标评价古代皇帝的综合排名系统，包含美观的Web前端界面。

## 功能特点

- 🏆 **综合评分系统**：基于11个维度的加权评分
- 🔍 **智能搜索**：支持皇帝姓名搜索，带自动补全建议
- 📊 **详细展示**：显示每项评分、权重和总分
- 📜 **历史信息**：提供皇帝简介和生平事迹
- 🎨 **现代UI**：美观的渐变设计和响应式布局

## 使用方法

### 方式一：直接打开HTML文件

1. 双击 `index.html` 文件，在浏览器中打开
2. 在搜索框输入皇帝姓名（如：唐太宗、秦始皇）
3. 点击"搜索"按钮或按回车键
4. 查看详细的评分信息和排名

### 方式二：使用本地服务器（推荐）

如果遇到跨域问题，可以使用本地服务器：

```bash
# 使用Python 3
python -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000

# 或使用Node.js的http-server
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 文件说明

- `index.html` - 主页面结构
- `style.css` - 样式文件
- `data.js` - 皇帝数据和权重配置
- `app.js` - 交互逻辑和功能实现

## 添加新皇帝

在 `data.js` 文件的 `EMPERORS` 数组中添加新对象：

```javascript
{
    name: "皇帝姓名",
    scores: {
        "德": 分数,
        "智": 分数,
        // ... 其他指标
    },
    intro: "简介",
    bio: "生平事迹",
}
```

系统会自动计算总分并更新排行榜。
