# TabBar 图标说明

请在 `images` 目录下放置以下图标文件：

## 需要的图标文件

1. **算分页面图标**
   - `calculator.png` - 未选中状态的图标（灰色）
   - `calculator-active.png` - 选中状态的图标（蓝色）

2. **历史页面图标**
   - `history.png` - 未选中状态的图标（灰色）
   - `history-active.png` - 选中状态的图标（蓝色）

## 图标规格要求

- **格式**: PNG（支持透明背景）
- **尺寸**: 81px × 81px（推荐）
- **颜色**:
  - 未选中: 灰色 (#7A7E83)
  - 选中: 蓝色 (#4A90E2)

## 图标建议

- **算分图标**: 可以使用计算器、加减号、或算盘等图标
- **历史图标**: 可以使用时钟、列表、或文档等图标

## 图标资源

您可以从以下网站获取免费图标：
- [IconFont](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)

下载后请重命名为对应的文件名并放置在 `images` 目录下。
